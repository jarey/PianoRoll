Piano Roll Help

This is a web based "piano roll" that allows you to create music by adding and editing notes graphically

Button/Shortcut Function
Space Play/Pause
Left/Right/Up/Down arrows Navigation
Shift + Left/Right/Up/Down Zomm In/Out Horizontally/Vertically
S (or pointer icon) Select and move notes
P (or pencil tool) Add/edit notes. Click to add a note and drag while holding to edit note length
Delete Deletes the selected note

Created by Larry Wang

Browser not supported

Unfortunately, this browser does not support the necessary Web Audio API features. Google Chrome is recommended for best performance. Firefox works but with lower sound quality.

Google Chrome recommended

Although this app will work on your browser, the audio quality may be inferior. Google Chrome is recommended
